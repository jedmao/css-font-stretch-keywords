# css-font-stretch-keywords

List of CSS [font stretches](https://developer.mozilla.org/en-US/docs/Web/CSS/font-stretch):
- normal
- condensed
- semi-condensed
- extra-condensed
- ultra-condensed
- expanded
- semi-expanded
- extra-expanded
- ultra-expanded

## Installation

```
npm install css-font-stretch-keywords [--save[-dev]]
```

## Usage

```js
var cssFontStretches = require('css-font-stretch-keywords');
cssFontStretches; /*
[
	"normal",
	"condensed",
	"semi-condensed",
	"extra-condensed",
	"ultra-condensed",
	"expanded",
	"semi-expanded",
	"extra-expanded",
	"ultra-expanded"
]
*/
```
